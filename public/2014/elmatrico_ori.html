Why all programming competitions are in the hands of CS guys? Why not people from other department...?
Have you been waiting for such a competition? Now the SEARCH is over, this ηvision, we present you EL MATRICO, which is an online MATLAB programming competition.




  •  There will be two problem statements, the deadline will be specified in the problem statement itself.

  •  The answers should be mailed to ee12b1028@iith.ac.in and the subject should contain "EL_MATRICO_Answer_Submission"

  •  The first set of problems will test your basic knowledge in MATLAB. The second set will application oriented.

  •  Participants are required to submit the answers in the following format

                      if the question requires an input, the participant should write a function that 
                  can take the input from the judges and should return the answer.
 
                  for eg:- If you are asked to find the sum of two numbers, you should submit the answer 
                  in the following format and submit the m file.

                  function [A] = find_sum(x,y)
 
                  A = x+y;
 
                  end
  •  All the answers should be in a single zipped file.

  •  A participant can only submit once. In case of multiple submissions only the first entry will be considered.

  •  Participants are allowed to use inbuilt MATLAB functions

  •  All the problems have equal weightage and it is not compulsory to solve all the problems.

  •  Decision of the judges will be final in all cases.

The judging will be based upon the time efficiency of your code. Faster code will get an edge over the others.


Prizes worth INR 3000/-
