About


ηvision, IIT Hyderabad in association with ABLab Solutions is here with a workshop on project processing machinery!

In this workshop you'll learn all about a reactor and an agitator, various types of failures that an agitator, reactor or a gearbox face in generaland how to overcome them using various tools like Fish Bone Diagram and Failure Mode Effect Analysis.

You'll also get to implement all the knowledge you've acquired in a real life case study!



Venue: IIT Hyderabad

Date: 7th & 8th January

Timing: 10:00 AM - 5:30 PM

Fee per person: ₹ 1350

Register Now

Workshop Benefits & Highlights


  • Clear your fundamentals of Industrial Reactors, Agitators; and Agitation and Mixing Process.
  • Know details of the parts of the agitator as well as conceptual knowledge about its design.
  • Know various modes of failures and their causes and preventions with the help of various tools like RCA (Root Cause Analysis), FMEA (Failure Mode Effect Analysis), Fish Bone Method and Kaizen Approach.
  • Implement all the knowledge acquired in a real life case study.
  • Get trained by experts
  • Certificate from ηvision, IIT Hyderabad
  • Live Interactive Question & Answer sessions
  • Get useful Materials
  • Learn
    • Knowledge about a reactor and an agitator
    • Knowledge about impellers, baffles and other parts constituting an industrial reactor as a whole.
    • Details of Operation and Maintenance Practices carried out in a Reactor
    • Various types of failures
    • Knowledge about various tools like RCA, Why-Why Analysis and Fish Bone Analysis
    • Implementation of these tools in real life failures caused in the reactors.
    • Kaizen Methods
    • How to finally record data using the same using Kaizen Methodology.
    • Implementation of all the above taught- in form of a case study.



Workshop Outline


  • What is a Reactor and an Agitator? What are the design concepts for various liquid, gas motions?
  • What is the difference between Agitation and Mixing?
  • What is an Impeller? What are its types?
  • What is a Baffle? What is its significance?
  • Operation and Maintenance of an Industrial Reactor
  • Various kinds of failures of Industrial Reactors
  • How to find out the reasons behind various modes of failures?
  • How to carry out the preventive and pro-active maintenances by analyzing the same with the help of various tools like Root Cause Analysis, Failure Mode Effect Analysis etc?
  • Implementing all the above learnt in a practical case study.
